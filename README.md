# 游戏设计-俄罗斯方块
## 设计目标
使用 qt6 配合进行俄罗斯方块游戏的设计，实现了快速下降，一键落地，形状预览，暂停等功能。此工程基于去年计算思维导论课后
自己完成的基于 C GDI 的[俄罗斯方块](https://github.com/snow-scald/cTetris.git)以及 [QT官方文档](https://doc.qt.io/)。
